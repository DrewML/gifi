# gifi

<img src="media/motivation.png" width="584">


## Requirements

- OS X
- [iTerm 2.9.x](https://www.iterm2.com/downloads.html)


## Installation

```
$ npm install gifi --global
```


## Usage

Replace `npm` with `gifi` and you're done:

```
$ gifi install koa --save
```
